# API Testing (Postman)

This folder contains API testing examples created using Postman.

The collections demonstrate how REST APIs can be tested using different scenarios such as happy path workflows and invalid input testing.

---

## Grocery Store API

The collection simulates a full workflow of an online grocery store:

* checking API status
* retrieving products
* creating a cart
* adding products to the cart
* updating quantities
* creating an order
* retrieving and managing orders

The collection also includes negative test scenarios to verify how the API handles invalid inputs.

---

## Tools Used

* Postman
* REST APIs
* JSON responses
